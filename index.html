Quantum Security Framework

Desenvolvido para Red Teams exigentes

O Cascavel é um framework de segurança ofensiva modular e extensível, com 84 plugins e integração com ferramentas OSINT avançadas como Shodan, Katana, Nuclei e mais. Automatize pentests de alta complexidade com relatórios completos em Markdown.

Instalação

git clone https://github.com/glferreira-devsecops/Cascavel.git
cd Cascavel
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

Execução

python3 cascavel.py -t exemplo.com
python3 cascavel.py --list-plugins
python3 cascavel.py --check-tools

Recursos

  • 🔌 84 plugins de segurança (rede, web, cloud, OSINT, auth, wireless, Red Team Elite 2026)
  • 🛠️ Integração com Nmap, Subfinder, Amass, Katana, Shodan e mais
  • 📁 Wordlists e templates prontos para Nuclei
  • 📝 Geração automática de relatórios em Markdown
  • 🖥️ CLI completa com argparse (modo interativo e batch)

Zero rastreamento. Zero coleta. Apenas código seguro, estático e auditável.
