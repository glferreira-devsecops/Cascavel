Quantum Security Framework

Built for Red Teams and advanced pentesting

Cascavel is a modular and extensible offensive security framework with 60 plugins designed to automate high-complexity pentesting workflows. Plugin-based architecture with native OSINT integrations (Shodan, Katana, Nuclei) and automated Markdown reporting.

Quick Start

git clone https://github.com/glferreira-devsecops/Cascavel.git
cd Cascavel && python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python3 cascavel.py -t target.com

Features

  • 🔌 30 security plugins (network, web, cloud, OSINT, auth, wireless)
  • ⚙️ Integration with Nmap, Subfinder, Amass, Katana, Shodan and more
  • 📁 Built-in wordlists and Nuclei vulnerability templates
  • 📝 Automatic Markdown reporting with timestamps
  • 🖥️ Full CLI with argparse (interactive + batch modes)
